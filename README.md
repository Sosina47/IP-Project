# Company Application Acceptance and Announcement System

A simple web-based platform that allows users to apply for job positions and enables the company to review, accept, or reject applications. Accepted applicants are displayed on a public announcements page.

## Features

### User Side
- **Home Page** – Introduction and navigation  
- **Job Application Page** – Submit applications online  
- **Application Status Page** – Check the progress of submitted applications  
- **Announcements Page** – View accepted applicants and company updates  
- **Contact Page** – Reach out to the company for inquiries

### Admin Side (Concept)
- Review submitted applications
- Accept or reject applicants
- Update announcements

## Technologies Used
- HTML
- CSS
- JavaScript
